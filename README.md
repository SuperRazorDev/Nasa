# NASA Project

## Instructions

Install:

```bash
npm install
```

or

```bash
yarn
```

Running:

```bash
npm run dev
```

or

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

### Highlights

- NASA API Eror handling with empty array
- Preview of uploaded images
- Extra upload step to confirm
- Scrolling by mouse wheel on carousel
- Caching of the images usisng useReducer

### Missing

- Mobile responsiveness
